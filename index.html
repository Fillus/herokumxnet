Heroku | Microsoft .NET Core Buildpacks

This repo is a very simple demonstration of running an ASP.NET Core web site on Heroku. This uses Visual Studio code and the ASP.NET Core Buildpack for Heroku and can be fully customized once deployed.

Welcome to .NET Core on Heroku.

This project is dedicated to running .NET Core and ASP.NET Core on Heroku using custom buildpacks. To begin, click the following Heroku Button to deploy a very basic ASP.NET Core app, built in Visual Studio, to Heroku.

Who owns the Project and Buildpacks?

Softtrends is the Heroku partner who owns and is maintaining the .NET Core and ASP.NET Core buildpacks for Heroku. For more details, check out www.softtrends.com or contact the project owner at samant@softtrends.com
